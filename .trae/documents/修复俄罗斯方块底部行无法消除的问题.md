# 修复俄罗斯方块底部行无法消除的问题

## 问题诊断

经过详细代码分析，发现了**两个关键问题**：

### 问题1：行消除时序错误（主要问题）

**位置：** [game.js](file:///d:/GitRepository/games/RussianSquare/js/game.js#L125-L147) 的 `_lockPiece` 方法

**当前代码：**
```javascript
_lockPiece() {
  if (!this.piece) return;
  this.board.lockPiece(this.piece);
  audio.play('lock');

  const fullLines = this.board.getFullLines();
  if (fullLines.length > 0) {
    this.renderer.triggerClearFlash(fullLines);
    if (fullLines.length === 4) {
      audio.play('tetris');
      this.renderer.triggerShake();
    } else {
      audio.play('lineClear');
    }
    const cleared = fullLines.length;
    setTimeout(() => {
      this.board.clearLines(fullLines);
      this.scoring.awardLines(cleared);
    }, 150);
  }

  this.spawnPiece();  // ❌ 问题：立即生成新方块，不等待消除完成
}
```

**问题分析：**
1. 当检测到完整行时，代码设置 150ms 延迟来播放消除动画
2. 但 `spawnPiece()` 在 setTimeout 之外**立即执行**
3. 新方块会立即生成，而此时旧方块还占据着位置
4. 150ms 后行消除才执行，但新方块已经在错误的状态下了

**对底部行的影响：**
- 当最底部行（索引 21）被填满时
- 新方块立即生成在顶部
- 150ms 后执行 `clearLines([21])`，删除索引 21 的行，顶部插入空行
- 所有行向下移动，但新方块的状态可能已经不同步
- 在某些情况下，底部行可能无法正确消除或显示异常

### 问题2：渲染偏移计算（潜在问题）

**位置：** [renderer.js](file:///d:/GitRepository/games/RussianSquare/js/renderer.js#L164-L170) 的 `_drawClearFlash` 方法

消除动画使用的行索引直接来自 `board.grid`，计算 `screenY = (row - HIDDEN_ROWS) * CELL_SIZE` 是正确的，但由于时序问题，可能导致闪烁效果显示异常。

## 修复方案

### 修复 `_lockPiece` 方法的时序问题

将 `spawnPiece()` 调用移到 `setTimeout` 内部，确保行消除完成后再生成新方块：

**修复后的代码：**
```javascript
_lockPiece() {
  if (!this.piece) return;
  this.board.lockPiece(this.piece);
  audio.play('lock');

  const fullLines = this.board.getFullLines();
  if (fullLines.length > 0) {
    this.renderer.triggerClearFlash(fullLines);
    if (fullLines.length === 4) {
      audio.play('tetris');
      this.renderer.triggerShake();
    } else {
      audio.play('lineClear');
    }
    const cleared = fullLines.length;
    setTimeout(() => {
      this.board.clearLines(fullLines);
      this.scoring.awardLines(cleared);
      this.spawnPiece();  // ✅ 修复：等待消除完成后再生成新方块
    }, 150);
  } else {
    this.spawnPiece();  // 没有需要消除的行，立即生成新方块
  }
}
```
